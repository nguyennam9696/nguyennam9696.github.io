Arrays vs. Hashes FIGHT!

Simply put, arrays and hashes are part of the Ruby Computer Language. However, there are key differences. Arrays are ordered lists of things. Easy enough? These "things" are known as "objects" in ruby. Like a noun. Hashes are a group of key-value pair. It is also a list of things. The main difference between arrays and hashes is that arrays is a list for integers and hashes is a list for any arbitrary objects or nouns :). If it is integers it is best to use arrays.
